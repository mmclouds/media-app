## 上下文
需要在图片生成面板中新增 z-image 的配置表单，并在积分系统内配置固定价格。

## 目标 / 非目标
- 目标：支持 z-image 的参数输入、请求映射、积分估算与模型注册。

## 决策
- 复用现有 `PromptEditor` 与 `AspectRatioField` 组件，保持样式与交互一致。

## 风险 / 权衡
